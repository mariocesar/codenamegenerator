# namegenerator
A Name generator of the form "Adjetive Noun"

Test it by

    $ python -m namegenerator
    loving fermi
    sick engelbart
    hopeful babbage
    distracted sinoussi
    modest swartz
    compassionate yalow
    modest pike
    furious babbage

or

    from namagenerator import get_random_name
    print(get_random_name)
